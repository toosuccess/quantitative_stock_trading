# Checklist

## 功能验证
- [x] 评分技能可以从数据库读取股票池股票
- [x] 评分技能可以正确计算技术指标（均线、MACD、布林线、OBV等）
- [x] 评分技能可以按照评分规则计算各指标得分
- [x] 评分技能可以计算综合得分并输出评级
- [x] 评分记录可以正确存储到`score_record`表
- [x] 同一天同一股票的评分记录可以正确更新

## 评分规则验证
- [x] 均线系统评分正确（最高25分）
- [x] 成交量评分正确（最高25分）
- [x] 趋势指标评分正确（最高20分）
- [x] 资金指标评分正确（最高15分）
- [x] 布林线评分正确（最高15分）
- [x] 综合得分计算正确（满分100分）

## 评级验证
- [x] 90-100分输出"强烈推荐"
- [x] 70-89分输出"推荐"
- [x] 50-69分输出"中性"
- [x] 30-49分输出"观望"
- [x] 0-29分输出"不推荐"

## 数据库验证
- [x] 评分记录包含股票代码、股票名称、评分日期
- [x] 评分记录包含各技术指标得分
- [x] 评分记录包含技术指标数值
- [x] 评分记录包含综合得分和评级

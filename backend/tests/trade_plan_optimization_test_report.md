## 功能验证

### 执行流程验证

```
1. 交易计划页面
   - 待执行计划显示"执行"按钮
   - 执行中计划显示"查看执行"按钮
   - 执行中计划禁用"编辑"和"删除"按钮

2. 点击执行
   - 弹出确认对话框
   - 显示执行说明

3. 确认执行
   - 计划状态变为"执行中"
   - 自动创建执行步骤
   - 跳转到交易执行页

4. 交易执行页面
   - 显示执行步骤列表
   - 可执行单个步骤
   - 支持筛选功能
```

### 删除流程验证

```
1. 点击删除按钮
   - 弹出确认对话框

2. 确认删除
   - 调用删除API
   - 列表自动刷新

3. 删除限制
   - 执行中的计划禁用删除按钮
   - 后端返回400错误
```

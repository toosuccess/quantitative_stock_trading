# 数据库设计说明

## 概述

数据库使用 SQLite，存储股票基本信息、评分记录、交易计划、交易记录等数据。

## 数据库文件

- **路径**: `backend/database/trading_system.db`
- **类型**: SQLite
- **编码**: UTF-8

## 表结构

### 一、stock_basic_info（股票基本信息表）

存储股票的基本信息，由选股技能填充。

| 字段名 | 类型 | 说明 | 约束 |
|--------|------|------|------|
| id | INTEGER | 主键 | PRIMARY KEY |
| stock_code | VARCHAR(20) | 股票代码 | NOT NULL |
| stock_name | VARCHAR(50) | 股票名称 | NOT NULL |
| stock_abbr | VARCHAR(20) | 股票简称 | - |
| exchange | VARCHAR(10) | 交易所（SH/SZ/BJ） | NOT NULL |
| industry | VARCHAR(100) | 所属行业 | - |
| sector | VARCHAR(100) | 所属板块 | - |
| list_date | DATE | 上市日期 | - |
| total_shares | BIGINT | 总股本 | - |
| float_shares | BIGINT | 流通股本 | - |
| market_cap | DECIMAL(20,2) | 总市值 | - |
| float_market_cap | DECIMAL(20,2) | 流通市值 | - |
| pe_ratio | DECIMAL(10,4) | 市盈率 | - |

| dividend_yield | DECIMAL(10,4) | 股息率 | - |

| create_time | DATETIME | 创建时间 | - |
| update_time | DATETIME | 更新时间 | - |
| remark | TEXT | 备注 | - |

**索引**：
- `stock_code`（唯一索引）

## 表关系图

```
stock_basic_info（股票基本信息）
    ↓
score_record（评分记录）
    ↓
trade_plan（交易计划）
    ↓
trade_execution_step（交易执行步骤）
    ↓
trade_record（交易记录）
    ↓
review_record（复盘记录）

account_info（账户信息）
    ↓
trade_plan（交易计划）
    ↓
trade_execution_step（交易执行步骤）
    ↓
trade_record（交易记录）
```

### 评级枚举值

| 值 | 说明 |
|------|------|
| 强烈推荐 | 综合评分90-100分 |
| 推荐 | 综合评分70-89分 |
| 中性 | 综合评分50-69分 |
| 观望 | 综合评分30-49分 |
| 不推荐 | 综合评分0-29分 |

### 交易类型枚举值

| 值 | 说明 |
|------|------|
| 买入 | 买入交易 |
| 卖出 | 卖出交易 |

### 状态枚举值

| 值 | 说明 |
|------|------|
| 待执行 | 等待执行 |
| 执行中 | 正在执行 |
| 已完成 | 执行完成 |
| 已取消 | 已取消 |

# 评分规则修改计划

## 一、需求概述

用户提出6项修改需求：
1. 政策面、消息面改为加减分项，不作为100分占比
2. 去掉龙头股限制，改为标签显示
3. 修复基本面指标数据为空的问题
4. 股票池列表添加最近打分时间，去掉打分次数
5. 菜单中账号管理放最后
6. 基本面评分规则中添加各指标打分规则说明

## 二、详细修改方案

### 1. 政策面、消息面改为加减分项

**修改文件**: `skills/multi_dimension_scoring.py`

**当前公式**:
```python
综合评分 = 技术面×0.3 + 基本面×0.3 + 消息面×0.2 + 政策面×0.2 - 减项扣分
```

**修改后公式**:
```python
综合评分 = 技术面×0.5 + 基本面×0.5 + 消息面加分 + 政策面加分 - 减项扣分
```

**加减分规则**:
| 维度 | 加分条件 | 分值 |
|------|---------|------|
| 消息面 | 利好消息 | +5~15分 |
| 消息面 | 中性消息 | 0分 |
| 消息面 | 负面消息 | -5~-15分 |
| 政策面 | 政策支持 | +10~30分 |
| 政策面 | 无政策 | 0分 |
| 政策面 | 政策利空 | -10~-20分 |

### 2. 去掉龙头股限制，改为标签显示

**修改文件**: `skills/stock_selection_skill.py`

**当前逻辑**: 非龙头股直接跳过
```python
if not meet_industry_leader_criteria(stock, industry):
    continue
```

**修改后逻辑**: 所有股票都参与评分，龙头股添加标签
```python
is_leader = meet_industry_leader_criteria(stock, industry)
stock['is_leader'] = is_leader
```

**前端显示**: 股票名称前添加"龙头"标签
```html
<el-tag v-if="scope.row.is_leader" type="danger" size="small">龙头</el-tag>
{{ scope.row.stock_name }}
```

### 3. 修复基本面指标数据为空问题

**问题分析**:
- `ak.stock_financial_analysis_indicator()` 接口返回数据不稳定
- 需要添加备用数据源

**修改方案**:
1. 优先使用 `ak.stock_financial_analysis_indicator()`
2. 备用使用 `ak.stock_yjyg_em()` (业绩预告)
3. 备用使用 `ak.stock_financial_report_sina()` (新浪财务数据)

### 4. 股票池列表添加打分时间

**修改文件**: `frontend/src/views/StockSelection.vue`

**当前列**:
```
评分次数 | 操作
```

**修改后列**:
```
最近打分时间 | 操作
```

### 5. 菜单中账号管理放最后

**修改文件**: `frontend/src/components/Layout.vue`

**当前菜单顺序**:
```
首页 | 股票池 | 账号管理 | 交易计划 | 交易执行 | 复盘分析
```

**修改后菜单顺序**:
```
首页 | 股票池 | 交易计划 | 交易执行 | 复盘分析 | 账号管理
```

### 6. 基本面评分规则添加打分规则说明

**修改文件**: `frontend/src/views/ScoreDetail.vue`

**当前显示**:
```
市盈率PE（15分）：估值合理性
```

**修改后显示**:
```
市盈率PE（15分）：
  - PE < 20：15分（估值合理）
  - PE 20-30：10分（估值适中）
  - PE 30-50：5分（估值偏高）
  - PE > 50：0分（估值过高）
```

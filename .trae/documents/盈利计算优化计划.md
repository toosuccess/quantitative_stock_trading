# 盈利计算优化计划

## 需求分析

### 当前问题

当前盈利计算逻辑：

```python
plan['profit'] = sell_amount - plan['buy_amount']
```

问题：

* 如果只有买入没有卖出，盈利显示为负的买入金额（不合理）

* 没有考虑当前持仓的浮动盈亏

### 需求

1. **已卖出**：盈利 = 卖出金额 - 买入金额
2. **未卖出（持仓中）**：浮动盈亏 = (当前价格 - 买入均价) × 持仓数量
3. **部分卖出**：浮动盈亏 = (当前价格 - 买入均价) × 剩余持仓数量 + 已实现盈亏

***

## 计算逻辑

### 盈利计算公式

```
持仓数量 = 买入数量 - 卖出数量
买入均价 = 买入总金额 / 买入总数量

情况1：持仓数量 = 0（已清仓）
  盈利 = 卖出总金额 - 买入总金额

情况2：持仓数量 > 0（持仓中）
  浮动盈亏 = (当前价格 - 买入均价) × 持仓数量
  已实现盈亏 = 卖出总金额 - (卖出数量 × 买入均价)
  总盈利 = 浮动盈亏 + 已实现盈亏
```

### 数据来源

* 当前价格：从score\_record表获取最新收盘价

* 买入/卖出数据：从trade\_record表获取

***

#### 1. 添加获取当前价格的方法

```python
def get_current_price(self, stock_code: str) -> float:
    """获取股票当前价格（最新收盘价）"""
    cursor = self.conn.cursor()
    cursor.execute('''
        SELECT close_price FROM score_record 
        WHERE stock_code = ?
        ORDER BY score_date DESC LIMIT 1
    ''', (stock_code,))
    row = cursor.fetchone()
    return row['close_price'] if row else 0
```

#### 2. 修改盈利计算逻辑

```python
def calculate_plan_profit(self, plan: Dict) -> Dict:
    """计算交易计划盈利"""
    buy_quantity = plan.get('buy_quantity', 0)
    buy_amount = plan.get('buy_amount', 0)
    sell_quantity = plan.get('sell_quantity', 0)
    sell_amount = plan.get('sell_amount', 0)
    stock_code = plan.get('stock_code', '')
    
    # 持仓数量
    holding_quantity = buy_quantity - sell_quantity
    
    # 买入均价
    buy_avg_price = buy_amount / buy_quantity if buy_quantity > 0 else 0
    
    if holding_quantity == 0:
        # 已清仓：实际盈亏
        profit = sell_amount - buy_amount
        profit_type = 'realized'
    else:
        # 持仓中：浮动盈亏
        current_price = self.get_current_price(stock_code)
        if current_price == 0:
            current_price = buy_avg_price  # 如果没有当前价格，使用买入均价
        
        # 浮动盈亏
        unrealized_profit = (current_price - buy_avg_price) * holding_quantity
        
        # 已实现盈亏（卖出部分）
        realized_profit = sell_amount - (sell_quantity * buy_avg_price)
        
        profit = unrealized_profit + realized_profit
        profit_type = 'unrealized'
    
    return {
        'profit': round(profit, 2),
        'profit_type': profit_type,
        'holding_quantity': holding_quantity,
        'buy_avg_price': round(buy_avg_price, 2)
    }
```

## 预期效果

### 已清仓计划

```
计划：买入华胜天成
买入：1000股 × 9.48 = 9480元
卖出：1000股 × 10.50 = 10500元
持仓：0股
盈利：10500 - 9480 = +1020元（已实现）
```

### 持仓中计划

```
计划：买入兆易创新
买入：1000股 × 22.00 = 22000元
卖出：1000股 × 55.00 = 55000元（已卖出）
持仓：0股
盈利：55000 - 22000 = +33000元（已实现）
```

### 部分卖出计划

```
计划：买入某股票
买入：1000股 × 10.00 = 10000元
卖出：500股 × 12.00 = 6000元
持仓：500股
当前价格：11.00元
买入均价：10.00元

浮动盈亏：(11.00 - 10.00) × 500 = +500元
已实现盈亏：6000 - (500 × 10.00) = +1000元
总盈利：500 + 1000 = +1500元
```

# 前端功能说明

## 概述

前端基于 Vue 3 + Element Plus 构建，提供股票池管理、评分详情、交易计划等功能。

## 技术栈

- **框架**: Vue 3
- **UI组件库**: Element Plus
- **状态管理**: Vue Composition API
- **HTTP客户端**: Axios
- **路由**: Vue Router
- **构建工具**: Vite

## 页面结构

```
frontend/
├── src/
│   ├── views/
│   │   ├── Home.vue           # 首页
│   │   ├── StockSelection.vue # 股票池页面
│   │   ├── ScoreDetail.vue    # 评分详情页面
│   │   ├── TradePlan.vue      # 交易计划页面
│   │   ├── TradeExecute.vue   # 交易执行页面
│   │   ├── Review.vue         # 复盘分析页面
│   │   └── AccountManage.vue  # 账号管理页面
│   ├── components/            # 公共组件
│   ├── api/                   # API接口
│   └── router/                # 路由配置
```

## 一、首页 (Home.vue)

### 功能概述
展示系统概览信息，包括股票池统计、最近评分记录等。

### 页面元素
- 系统标题和日期显示
- 股票池统计卡片
- 快速导航入口

## 二、股票池页面 (StockSelection.vue)

### 功能概述
展示股票池列表，支持筛选、排序、批量评价等功能。

### 2.1 统计卡片区域

| 卡片 | 功能 | 交互 |
|------|------|------|
| 股票总数 | 显示股票池总数 | 无 |
| 强烈推荐 | 显示强烈推荐股票数量 | 点击筛选强烈推荐股票 |
| 推荐 | 显示推荐股票数量 | 点击筛选推荐股票 |
| 中性 | 显示中性的股票数量 | 点击筛选中性股票 |
| 观望 | 显示观望股票数量 | 点击筛选观望股票 |
| 不推荐 | 显示不推荐股票数量 | 点击筛选不推荐股票 |

### 2.2 股票列表表格

| 列名 | 说明 | 功能 |
|------|------|------|
| 复选框 | 选择股票 | 支持多选 |
| 股票代码 | 股票代码 | - |
| 股票名称 | 股票名称 | - |
| 收盘价 | 最新收盘价 | - |
| 综合评分 | 综合评分+趋势图 | 显示评分和股价趋势 |
| 评级 | 评级标签 | 带颜色显示 |
| 各维度评分 | 技术面/基本面/消息面/政策面/扣分 | 支持排序，悬停显示详情 |
| 综合评价 | 评价摘要 | - |
| 最近打分时间 | 最后评分时间 | - |
| 操作 | 查看详情/创建计划 | 按钮操作 |

### 2.3 综合评分趋势图

- **评分趋势线**：绿色/红色/灰色，根据评分变化显示
- **股价趋势线**：蓝色半透明，显示股价变化
- **数据来源**：最近10次评分记录

### 2.4 批量评价功能

- 点击"批量评价"按钮启动
- 显示进度对话框
- 逐个评价选中的股票
- 显示当前进度（如：1/10）
- 显示正在评价的股票名称
- 评价完成后自动刷新列表

### 2.5 筛选和排序

- **评级筛选**：点击统计卡片快速筛选
- **维度排序**：点击表头按技术面、基本面、消息面、政策面、扣分排序
- **综合排序**：按综合评分排序

## 三、评分详情页面 (ScoreDetail.vue)

### 功能概述
展示单只股票的详细评分信息，包括各维度明细、历史记录等。

### 3.1 评分概览区域

| 元素 | 说明 |
|------|------|
| 股票代码/名称 | 基本信息 |
| 评分次数 | 历史评分次数 |
| 综合评分 | 大号数字显示 |
| 评级 | 带颜色标签 |
| 各维度得分 | 技术面/基本面/消息面/政策面/扣分 |

### 3.2 股票基本信息卡片

| 字段 | 说明 |
|------|------|
| 股票代码 | 股票代码 |
| 股票名称 | 股票名称 |
| 交易所 | 交易所（SH/SZ/BJ） |
| 行业 | 所属行业 |

| 市盈率PE | 市盈率 |
| 市净率PB | 市净率 |

| MACD | 得分 + 详情（MACD金叉: 是/否） |
| OBV | 得分 + 详情（OBV>MAOBV: 是/否） |
| 布林带 | 得分 + 详情（布林带中轨上方: 是/否） |

#### 基本面明细
| 指标 | 显示内容 |
|------|---------|
| 市盈率PE | 得分 + 数值 + 详情 |
| 市净率PB | 得分 + 数值 + 详情 |
| ROE | 得分 + 数值 + 详情 |
| 净利润增长率 | 得分 + 数值 + 详情 |
| 营收增长率 | 得分 + 数值 + 详情 |
| 负债率 | 得分 + 数值 + 详情 |

#### 消息面明细
| 列名 | 说明 |
|------|------|

| 类型 | 利好/利空/中性 |
| 事件标题 | 新闻标题 |
| 内容摘要 | 新闻摘要（前100字） |
| 得分影响 | 分数影响 |
| 操作 | 查看详情（跳转外部链接） |

#### 政策面明细
| 列名 | 说明 |
|------|------|
| 日期 | 政策日期 |
| 政策级别 | 国家政策/行业政策 |
| 政策名称 | 政策名称 |
| 政策内容 | 政策内容 |
| 得分影响 | 分数影响 |
| 操作 | 查看详情（如有链接） |

#### 减项扣分明细
- 显示风险项列表
- 无风险项时显示"该股票暂无明显风险项"

### 3.4 历史评分记录

| 列名 | 说明 |
|------|------|
| 评分时间 | 评分时间 |
| 综合 | 综合评分 |
| 评级 | 评级 |
| 技术面 | 技术面得分 |
| 基本面 | 基本面得分 |
| 消息面 | 消息面得分 |
| 政策面 | 政策面得分 |
| 扣分 | 减项扣分 |

### 3.5 评分变化趋势图

- 折线图展示历史评分变化
- X轴：日期

## 公共组件

### 评分标签组件
- 根据评分显示不同颜色
- 90+：绿色（success）
- 70-89：蓝色（primary）
- 50-69：橙色（warning）
- 0-49：灰色（info）

### 趋势图组件
- SVG迷你折线图
- 支持双线叠加（评分+股价）
- 自动计算坐标点

## API调用

### 请求配置
- 基础URL：`/api/v1`
- 超时时间：120秒
- 请求拦截：添加认证头

### 错误处理
- 网络错误提示
- 业务错误提示
- 超时处理

## 样式规范

### 颜色定义
```css
--success-color: #67c23a  /* 成功/上涨 */
--primary-color: #409eff  /* 主要/推荐 */
--warning-color: #e6a23c  /* 警告/中性 */
--danger-color: #f56c6c   /* 危险/下跌 */
--info-color: #909399     /* 信息/观望 */
```

# 评分详情页面打分明细展示计划

## 一、需求分析

### 1.1 当前问题
1. 评分详情页面只显示各维度总分，没有展示打分明细
2. 政策面、消息面没有具体的事件和时间节点
3. 基本面、技术面没有具体的指标和打分过程
4. 股票池列表各维度评分与评分详情不一致

### 1.2 需要实现的功能
1. **技术面明细**：展示均线、成交量、趋势、资金、布林线的具体指标值和打分过程
2. **基本面明细**：展示PE、PB、净利润增长、营收增长、ROE、负债率的具体值和打分过程
3. **消息面明细**：展示具体的事件内容、事件时间、事件来源
4. **政策面明细**：展示具体的政策内容、发布时间、政策级别
5. **减项明细**：展示具体的扣分项和原因

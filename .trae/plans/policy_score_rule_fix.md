# 政策面评分规则调整和明细显示修复计划

## 一、问题分析

### 1.1 政策面评分规则描述问题
前端显示的规则：
- 国家政策：+10~+30分
- 地方政策：+5~+20分
- 行业规划：+5~+20分
- 监管环境：-5~-20分

实际代码中的评分规则：
- 未来产业（数字经济、人工智能、核聚变、6G）：10分
- 战略性新兴产业（新能源、新材料、高端制造、半导体、生物医药）：8分
- 民生消费（消费升级）：5分
- 其他：3分

**需要调整前端描述与实际规则一致**

### 1.2 消息面明细和政策面明细为空问题

**根本原因**：数据库中存储的是Python字典格式（单引号），但前端JSON.parse需要双引号格式。

数据库存储格式：
```
{'score': 10, 'policies': [{'date': '2026-02-21', ...}]}
```

正确JSON格式：
```
{"score": 10, "policies": [{"date": "2026-02-21", ...}]}
```

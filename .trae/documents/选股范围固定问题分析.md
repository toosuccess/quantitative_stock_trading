**问题：**
- 每个行业只预设了8只股票
- 总共只有约80只股票可选

- 每次选股都在这个固定范围内选择

## 二、解决方案

### 方案A：动态获取全市场股票（推荐）

使用AkShare API动态获取股票列表，根据行业分类自动筛选：

```python
import akshare as ak

def get_industry_stocks(industry_name):
    """根据行业名称动态获取股票列表"""
    # 获取全市场股票行业分类
    df = ak.stock_board_industry_name_em()
    # 筛选指定行业的股票
    industry_stocks = df[df['板块名称'].str.contains(industry_name)]
    return industry_stocks['股票代码'].tolist()
```

### 方案B：扩大硬编码股票池

在现有基础上，为每个行业添加更多股票（每行业20-30只）

### 方案C：混合方案

1. 保留现有硬编码股票作为"核心股票池"
2. 动态获取补充股票作为"扩展股票池"
3. 优先分析核心股票池，再分析扩展股票池

## 三、推荐方案：动态获取

### 实现步骤

1. **获取全市场股票行业分类**
   - 使用 `ak.stock_board_industry_name_em()` 获取行业板块
   - 使用 `ak.stock_board_industry_cons_em(symbol)` 获取行业成分股

2. **根据政策导向行业匹配股票**
   - 将政策导向行业（新能源、半导体等）与AkShare行业分类对应
   - 动态获取每个行业的所有成分股

3. **应用筛选条件**
   - 剔除ST、退市风险股票
   - 剔除亏损股票
   - 筛选行业龙头

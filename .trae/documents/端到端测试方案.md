# 端到端测试方案

## 测试目标
基于东华软件(002065)和茅台(600519)两只股票，完整测试选股、评价、做计划、执行的流程。

## 测试股票
| 股票代码 | 股票名称 | 说明 |
|---------|---------|------|
| sz002065 | 东华软件 | 测试股票 |
| sh600519 | 贵州茅台 | 龙头股 |

## 测试场景

### 场景1：股票池列表页面（选股流程）
**测试步骤：**
1. 访问股票池列表页面
2. 验证股票列表显示正确
3. 验证各维度评分显示正确（技术面、基本面、消息面、政策面、扣分）
4. 验证消息面和政策面显示为加减分格式
5. 验证综合评价显示正确

**预期结果：**
- 股票列表显示东华软件和茅台
- 消息面显示格式为"消0"（加减分格式）
- 政策面显示格式为"政0"（加减分格式）

### 场景2：评分详情页面（评价流程）
**测试步骤：**
1. 点击"查看详情"按钮
2. 验证评价体系说明显示正确
3. 验证消息面和政策面无基准分说明
4. 验证各维度评分卡片显示正确
5. 验证综合评价内容正确
6. 验证历史评分记录表格显示正确

**预期结果：**
- 消息面说明显示"注：根据消息利好利空直接加减分，无基准分"
- 政策面说明显示"注：根据政策利好利空直接加减分，无基准分"
- 消息面和政策面显示为"加减分"格式

### 场景3：创建交易计划流程
**测试步骤：**
1. 返回股票池列表
2. 点击"创建计划"按钮
3. 填写交易计划信息
4. 提交创建计划
5. 验证计划创建成功

**预期结果：**
- 计划创建成功
- 显示计划详情

### 场景4：交易执行流程
**测试步骤：**
1. 访问交易计划页面
2. 查看创建的计划
3. 执行交易计划
4. 验证执行结果

**预期结果：**
- 交易执行成功
- 显示执行记录

## 测试数据验证点
1. 消息面评分：应为0（中性，无加减分）
2. 政策面评分：应为0（中性，无加减分）
3. 综合评分公式：技术面×0.5 + 基本面×0.5 + 消息面加减分 + 政策面加减分 - 减项扣分
4. 综合评价格式：支持正负分显示

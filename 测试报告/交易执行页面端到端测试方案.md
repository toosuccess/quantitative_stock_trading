### 5.2 发现的问题

#### 问题1：清仓逻辑代码错误
**描述**：`execute_step` 方法调用了不存在的 `get_holding_profit` 方法
**影响**：减仓/止盈减仓/止损清仓等操作后，计划状态不会自动更新为"已完成"
**修复**：将 `get_holding_profit` 改为 `get_holding_quantity`

**修复前代码**：
```python
holding_info = self.get_holding_profit(step['plan_id'], 0)
current_holding = holding_info.get('holding_quantity', 0)
```

**修复后代码**：
```python
current_holding = self.get_holding_quantity(step['plan_id'])
```

**验证结果**：
- 比亚迪：买入1000股，卖出1000股，持仓=0，状态=已完成 ✅
- 宁德时代：持仓=0，状态=已完成 ✅

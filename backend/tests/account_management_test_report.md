## 优化内容

### 1. 账号管理界面

| 功能 | 说明 |
|------|------|
| 账号列表 | 显示账号名称、类型、券商、总资产、可用资金、市值、盈亏 |
| 新增账号 | 支持添加实盘/模拟账号 |

| 账号汇总 | 查看账号统计信息 |

### 2. 交易计划关联证券账号

| 功能 | 说明 |
|------|------|
| 账号选择 | 创建计划时选择证券账号 |
| 账号筛选 | 按账号筛选交易计划 |
| 账号显示 | 列表显示账号名称 |

### 3. 复盘分析层级结构

| 层级 | 内容 |
|------|------|
| 第一层 | 证券账号（账号名称、总资产、盈亏、计划数） |
| 第二层 | 交易计划（计划名称、股票、盈亏、状态） |
| 第三层 | 交易记录（类型、方向、价格、数量、金额、日期、时间） |

### 4. 统计报表真实数据

| 指标 | 计算方式 |
|------|----------|
| 交易胜率 | 盈利交易次数 / 总交易次数 × 100% |
| 盈亏比 | 平均盈利金额 / 平均亏损金额 |
| 最大回撤 | 历史最高点后的最大跌幅 |
| 总交易次数 | 交易记录总数 |

## 功能验证

### 账号管理验证

```
功能：
- 新增账号：输入名称、类型、券商、资产信息
- 编辑账号：修改账号信息
- 删除账号：检查是否有关联计划，有则禁止删除
- 账号汇总：显示计划数、交易数、买卖总额、胜率、盈亏比等
```

### 交易计划关联验证

```
创建计划：
- 选择证券账号（下拉框）
- 输入股票代码、名称
- 自动计算止损止盈

计划列表：
- 显示账号名称列
- 支持按账号筛选
```

### 复盘分析层级验证

```
层级结构：
第一层：证券账号
  - 账号名称、总资产、盈亏、计划数
  - 点击展开显示计划

第二层：交易计划
  - 计划名称、股票、盈亏、状态
  - 点击展开显示交易记录

第三层：交易记录
  - 类型、方向、价格、数量、金额、日期、时间
  - 点击"详情"查看详细信息
```

### 统计数据验证

```
数据来源：
- 从trade_record表获取交易记录
- 从trade_plan表获取计划信息
- 从account_info表获取账号信息

计算逻辑：
- 胜率 = 盈利计划数 / 总计划数 × 100%
- 盈亏比 = 平均盈利金额 / 平均亏损金额
- 最大回撤 = 历史最高点后的最大跌幅
- 总交易次数 = 交易记录数
```

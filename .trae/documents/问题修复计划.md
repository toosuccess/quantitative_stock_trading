# 问题修复计划

## 一、问题分析

### 1. 最近打分时间为空
**原因**: API `/api/v1/stocks/pool/scores` 没有返回 `create_time` 字段

**解决方案**: 修改 `routes.py` 中的查询，添加 `create_time` 字段

### 2. 股票名称没有显示
**原因**: 部分股票的 `stock_name` 字段可能为空或格式问题

**解决方案**: 检查数据保存逻辑，确保股票名称正确保存

### 3. 龙头股标记没有显示
**原因**: API `/api/v1/stocks/pool/scores` 没有返回 `is_leader` 字段

**解决方案**: 修改 `routes.py` 中的查询，添加 `is_leader` 字段

### 4. 消息面和政策面规则需要修改
**原因**: 消息面和政策面已改为加减分项，但前端规则说明还是显示"满分100分"

**解决方案**: 修改 `ScoreDetail.vue` 中的规则说明，改为"加减分项"

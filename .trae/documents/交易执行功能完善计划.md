# 交易执行功能完善计划

## 需求分析

### 问题1：止盈清仓价格默认值
**当前情况**：
- 止盈清仓的成交价格默认为当前价格
**期望**：
- 默认等于执行步骤中交易方向为"止盈清仓"对应的目标价格

### 问题2：账号管理盈亏计算
**当前情况**：
- 盈亏是固定值0
**期望**：
- 盈亏需要根据实际交易变化

### 问题3：执行步骤目标价格记录
**当前情况**：
- 执行后目标价格显示为"-"
**期望**：
- 执行后需要记录目标价格

### 问题4：成交金额显示格式
**当前情况**：
- 成交金额只显示数字
**期望**：
- 买入时前面添加"-"
- 卖出时前面添加"+"

### 问题5：交易计划完成后按钮置灰
**当前情况**：
- 交易计划完成后，执行按钮仍可点击
**期望**：
- 交易计划状态为"完成"时，所有"执行"按钮置灰
- "添加执行计划"界面的"确认添加"按钮也置灰

## 实施方案

### 问题1修复：止盈清仓价格默认值

文件：`frontend/src/views/TradeExecution.vue`

```javascript
const showExecuteStepDialog = (row) => {
  const direction = row.trade_direction
  const currentPrice = row.current_price || row.target_price || 0
  
  // 止损清仓/止盈清仓使用目标价格
  let defaultPrice = currentPrice
  if (direction === '止损清仓' || direction === '止盈清仓') {
    defaultPrice = row.target_price || currentPrice
  }
  
  executeStepForm.value = {
    ...
    trade_price: defaultPrice,
    ...
  }
}
```

### 问题2修复：账号管理盈亏计算

文件：`backend/app/services/trading_manager.py`

在获取账户列表时，计算盈亏：
```python
# 盈亏 = 总资产 - 初始资产
profit_loss = account['total_assets'] - account.get('initial_assets', account['total_assets'])
```

或者根据交易记录计算：
```python
# 盈亏 = 卖出总额 - 买入总额
profit_loss = sell_amount - buy_amount
```

### 问题3修复：执行步骤目标价格记录

文件：`backend/app/services/trading_manager.py`

在execute_step方法中，更新步骤的目标价格：
```python
self.update_execution_step(step_id, {
    'target_price': trade_price,  # 记录实际成交价格
    ...
})
```

### 问题4修复：成交金额显示格式

文件：`frontend/src/views/TradeExecution.vue`

```vue
<el-table-column prop="trade_amount" label="成交金额" width="120">
  <template #default="scope">
    <span :class="scope.row.trade_type === '卖出' ? 'profit' : 'loss'">
      {{ scope.row.trade_type === '卖出' ? '+' : '-' }}{{ scope.row.trade_amount?.toFixed(2) }}
    </span>
  </template>
</el-table-column>
```

### 问题5修复：交易计划完成后按钮置灰

文件：`frontend/src/views/TradeExecution.vue`

```vue
<!-- 执行按钮 -->
<el-button 
  :disabled="isPlanCompleted(scope.row.plan_id)"
  @click="showExecuteStepDialog(scope.row)"
>
  执行
</el-button>

<!-- 确认添加按钮 -->
<el-button 
  type="primary" 
  :disabled="isCurrentPlanCompleted"
  @click="confirmAddStep"
>
  确认添加
</el-button>

<script>
const isPlanCompleted = (planId) => {
  const plan = allPlans.value.find(p => p.plan_id === planId)
  return plan?.status === 'completed'
}

const isCurrentPlanCompleted = computed(() => {
  const plan = allPlans.value.find(p => p.plan_id === addStepForm.value.plan_id)
  return plan?.status === 'completed'
})
</script>
```

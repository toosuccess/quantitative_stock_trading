## 架构调整内容

### 调整前架构
```
前端 → 后端API → 实时选股 → akshare获取数据
```

### 调整后架构
```
选股项目(独立) → 写入数据库 → 后端API读取 → 前端展示
```

### 核心变更

1. **后端API变更**
   - 新增 `GET /api/v1/scores` - 从数据库读取评分记录列表
   - 新增 `GET /api/v1/scores/{code}` - 从数据库读取指定股票评分
   - 移除 `GET /api/v1/select` - 选股接口（选股是另一个项目）
   - 移除 `GET /api/v1/stocks/score/{code}` - 实时评分接口

2. **前端页面变更**
   - Home.vue：从scoreApi获取评分股票数量
   - StockSelection.vue：改名为"评分结果"，从数据库读取评分记录
   - api/index.js：新增scoreApi，移除selectApi

# 基本面说明
## 基于国家宏观政策（如"十五五"规划）和部委最新政策动向，结合周期性热点窗口，筛选符合产业升级方向的行业龙头股票

## 1.自动收集和分析国家五年规划、工信部、国资委、商务部等部委的最新政策文件和会议信息
  - 政策窗口期：两会、半年会、年度经济工作会议等
  - 行业事件：技术突破、政策发布、行业峰会等
  - 市场规律：春季躁动、财报季、年末行情等
  - 新兴行业：具有快速成长潜力的行业
  - 高成长行业：ROE、ROA、净利润等指标都较高的行业
  - 国家政策：符合国家宏观政策的行业
  - 未来有增长潜力：未来1-3年有增长潜力的行业
  - 未来产业升级方向：符合国家宏观政策的产业升级方向

## 1.1 重点关注行业
  - 新能源：包括新能源汽车、光伏、风电、储能等
  - 新材料：包括稀土材料、半导体材料、先进金属材料、复合材料等
  - 6G：包括通信设备、运营商、网络安全、云计算等
  - 核聚变：包括核电设备、电力装备、新能源技术等
  - 消费升级：包括高端消费、医药健康、金融服务、食品饮料等
  - 半导体：包括芯片设计、制造、封装测试等
  - 人工智能：包括算力、算法、应用等
  - 生物医药：包括创新药、医疗器械、疫苗等
  - 高端制造：包括智能制造、工业机器人等
  - 数字经济：包括云计算、大数据、区块链等


## 2.选择行业龙头和业绩良好的上市公司
  - 不能是亏损公司，不能是ST股票
  - 不能是新上市公司
  - 未来没有股东减持计划，没有解禁
  - 没有定增计划


## 2.1 行业龙头股票筛选
  - 技术创新：具有最新技术突破的公司
  - 市场规模：在行业内排名前30%的公司
  - 未来有增长潜力：未来1-3年有增长潜力的公司


# 技术面说明
## 技术分析指标
- 均线系统：股价站稳20日均线且M20向上；60/180/250日均线多头排列 
- 成交量：当日成交量＞5日/60日均量；换手率5%-20%  
- 趋势指标：DMI（14，6）的+DI＞-DI且ADX/ADXR上升；MACD 指标： DIFF＞DEA>0
- 资金指标：OBV＞MAOBV且持续上行，资金净流入。
- 布林线（20，2）：股价位于布林线的上轨和中轨之间，且未超买（股价未超买：股价未超过上轨）。


## 技术分析工具
- **API接口**：使用 akshare 库获取股票价格数据
  - 支持函数：stock_zh_a_daily、stock_zh_index_daily、stock_zh_a_hist
  - 支持多种股票代码格式
- **数据处理**：
  - 使用 pandas 库处理和分析数据
  - 动态调整列名，处理不同数据源的格式差异
  - 转换日期格式和计算单位转换


# 输出说明
## 输出包括
 - 基本信息
 - 基本面：基本面分析报告
 - 技术面：技术分析报告
 - 政策面：政策受益分析报告，最近的政策时间窗口
 - 扣分项：退市警示、大额解禁、股东减持计划、商誉减值超净资产20%等
 - 综合得分：综合政策面得分，基本面分析得分和技术指标得分，给出综合得分。
  -- 综合得分：政策面得分*0.25 + 基本面分析得分*0.25 + 技术指标得分*0.5
  -- 给出各项得分理由：
    - 政策面得分：根据最近政策时间窗口判断股票是否受益于政策支持，给出得分。
    - 基本面分析得分：根据公司基本面指标判断股票是否有增长潜力，给出得分。
    - 技术指标得分：根据技术指标判断股票是否符合趋势和资金指标，给出得分。
    - 扣分项得分：根据股票是否存在退市警示、大额解禁、股东减持计划、商誉减值超净资产20%等情况，给出扣分。
 - 最近五天股价：最近5个交易日的股价数据，格式：YYYY年MM月DD日:XX元

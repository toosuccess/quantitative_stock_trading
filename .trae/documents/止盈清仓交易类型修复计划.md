# 止盈清仓交易类型修复计划

## 需求分析

### 问题描述
**当前情况**：
- 止盈清仓的交易类型显示为"买入"
- 期望：止盈清仓的交易类型应该是"卖出"

### 问题原因
在 `trading_manager.py` 的 `execute_step` 方法中：
```python
trade_type = '卖出' if step['trade_direction'] in ['减仓', '清仓'] else '买入'
```

这里只判断了'减仓'和'清仓'，没有包含'止损清仓'和'止盈清仓'。

---

## 实施方案

### 修复代码

文件：`backend/app/services/trading_manager.py`

```python
trade_type = '卖出' if step['trade_direction'] in ['减仓', '清仓', '止损清仓', '止盈清仓'] else '买入'
```

- 测试止损清仓的交易类型是否为"卖出"

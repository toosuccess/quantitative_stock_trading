# 评分系统设计说明

## 概述

评分系统是量化交易系统的核心模块，用于对股票进行多维度综合评价，输出评级和投资建议。

## 评分维度

评分系统包含五个维度：
1. **技术面评分**（满分100分）- 基于技术指标分析
2. **基本面评分**（满分100分）- 基于财务数据分析
3. **消息面评分**（加减分）- 基于新闻事件分析
4. **政策面评分**（加减分）- 基于行业政策分析
5. **减项扣分**（扣分）- 基于风险因素分析

## 一、技术面评分（满分100分）

### 1.1 均线系统（25分）

| 条件 | 得分 | 说明 |
|------|------|------|
| 股价站稳20日均线且M20向上 | 25分 | 均线多头排列，趋势向上 |
| 股价在20日均线上方但M20走平 | 15分 | 趋势不明朗 |
| 其他情况 | 0分 | 趋势向下或不明 |

### 1.2 成交量（25分）

| 条件 | 得分 | 说明 |
|------|------|------|
| 成交量放大且换手率5%-20% | 25分 | 量价配合良好 |
| 成交量放大但换手率过高或过低 | 15分 | 量价配合一般 |
| 成交量萎缩 | 0分 | 量价配合差 |

### 1.3 MACD（20分）

| 条件 | 得分 | 说明 |
|------|------|------|
| MACD金叉且在零轴上方 | 20分 | 强势上涨信号 |
| MACD金叉但在零轴下方 | 10分 | 弱势反弹信号 |
| MACD死叉 | 0分 | 下跌信号 |

### 1.4 OBV（15分）

| 条件 | 得分 | 说明 |
|------|------|------|
| OBV > MAOBV且持续上行 | 15分 | 资金持续流入 |
| OBV > MAOBV但走平 | 10分 | 资金流入放缓 |
| OBV < MAOBV | 0分 | 资金流出 |

### 1.5 布林带（15分）

| 条件 | 得分 | 说明 |
|------|------|------|
| 股价在中轨上方且未触及上轨 | 15分 | 上涨空间充足 |
| 股价触及上轨 | 10分 | 短期可能回调 |
| 股价在中轨下方 | 0分 | 弱势状态 |

## 二、基本面评分（满分100分）

### 2.1 市盈率PE（20分）

| PE范围 | 得分 | 说明 |
|--------|------|------|
| PE < 30 | 20分 | 估值适中 |
| 30 ≤ PE < 50 | 10分 | 估值偏高 |
| PE ≥ 50 或 PE ≤ 0 | 0分 | 估值过高或亏损 |

### 2.2 市净率PB（15分）

| PB范围 | 得分 | 说明 |
|--------|------|------|
| PB < 2 | 15分 | 资产价值高 |
| 2 ≤ PB < 3 | 10分 | 资产价值适中 |
| 3 ≤ PB < 5 | 5分 | 资产价值偏低 |
| PB ≥ 5 | 0分 | 资产价值低 |

### 2.3 净资产收益率ROE（20分）

| ROE范围 | 得分 | 说明 |
|---------|------|------|
| ROE > 15% | 20分 | 回报优秀 |
| 10% < ROE ≤ 15% | 15分 | 回报良好 |
| 5% < ROE ≤ 10% | 10分 | 回报一般 |
| ROE ≤ 5% | 0分 | 回报较差 |

### 2.4 净利润增长率（20分）

| 增长率范围 | 得分 | 说明 |
|-----------|------|------|
| 增长率 > 30% | 20分 | 盈利能力强 |
| 10% < 增长率 ≤ 30% | 10分 | 盈利稳定 |
| 增长率 ≤ 10% | 0分 | 盈利下滑 |

### 2.5 营收增长率（15分）

| 增长率范围 | 得分 | 说明 |
|-----------|------|------|
| 增长率 > 20% | 15分 | 业务扩张快 |
| 10% < 增长率 ≤ 20% | 10分 | 业务稳健 |
| 增长率 ≤ 10% | 0分 | 业务萎缩 |

### 2.6 负债率（10分）

| 负债率范围 | 得分 | 说明 |
|-----------|------|------|
| 负债率 < 50% | 10分 | 财务稳健 |
| 50% ≤ 负债率 < 70% | 5分 | 财务适中 |
| 负债率 ≥ 70% | 0分 | 财务风险高 |

## 三、消息面评分（加减分）

### 3.1 利好消息

| 事件类型 | 得分 | 示例 |
|---------|------|------|
| 行业利好 | +5~+15分 | 行业政策支持、行业景气度提升 |
| 公司利好 | +5~+15分 | 业绩预告、中标、合作 |
| 机构评级 | +5~+15分 | 券商研报评级、目标价 |
| 资金动向 | +5~+10分 | 北向资金、机构持仓变化 |

### 3.2 利空消息

| 事件类型 | 得分 | 示例 |
|---------|------|------|
| 负面新闻 | -5~-15分 | 业绩下滑、诉讼、监管处罚 |

## 四、政策面评分（加减分）

### 4.1 行业分类

| 行业类型 | 得分 | 示例行业 |
|---------|------|---------|
| 未来产业 | +10分 | 6G、核聚变、人工智能、数字经济 |
| 战略性新兴产业 | +8分 | 新能源、新材料、半导体、生物医药、高端制造 |
| 民生消费 | +5分 | 消费升级 |
| 其他行业 | +3分 | - |

## 五、减项扣分（扣分）

| 风险类型 | 扣分 | 说明 |
|---------|------|------|
| 高质押率 | -10分 | 质押比例超过50% |
| 减持公告 | -15分 | 大股东减持 |
| 诉讼风险 | -20分 | 重大诉讼 |
| 财务异常 | -15分 | 财务数据异常 |
| 限售解禁 | -10分 | 大量限售股解禁 |
| 负面新闻 | -10分 | 重大负面报道 |

## 六、综合评分计算

### 6.1 计算公式

```
综合评分 = 技术面 × 0.5 + 基本面 × 0.5 + 消息面加减分 + 政策面加减分 - 减项扣分
```

### 6.2 评级标准

| 评分范围 | 评级 | 投资建议 |
|---------|------|---------|
| 90-100分 | 强烈推荐 | 优质标的，可重点关注 |
| 70-89分 | 推荐 | 良好标的，可考虑介入 |
| 50-69分 | 中性 | 观望为主，等待机会 |
| 30-49分 | 观望 | 风险较大，谨慎参与 |
| 0-29分 | 不推荐 | 风险过高，避免参与 |

## 七、数据来源

| 数据类型 | 数据来源 | 接口/方法 |
|---------|---------|---------|
| 股票价格数据 | akshare | `stock_zh_a_hist` |
| 财务指标数据 | akshare | `stock_financial_analysis_indicator` |
| 个股信息 | akshare | `stock_individual_info_em` |
| 新闻数据 | akshare | `stock_news_em` |
| 行业数据 | 配置 | `POLICY_ORIENTED_INDUSTRIES` |

## 八、评分流程

```
1. 获取股票代码
2. 获取价格数据 → 计算技术面评分
3. 获取财务数据 → 计算基本面评分
4. 获取新闻数据 → 计算消息面评分
5. 获取行业数据 → 计算政策面评分
6. 检查风险因素 → 计算减项扣分
7. 计算综合评分 → 输出评级
8. 保存评分记录到数据库
```

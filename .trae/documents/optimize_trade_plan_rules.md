# 优化交易计划制定规则

## 需求分析

用户要求修改交易计划制定规则，包括：

### 一、仓位配置策略
1. **现金比例**：30%-40%（防御性底仓）
2. **股票仓位**：60%-70%（进攻性仓位）
   - 牛市上限可提至80%
   - 熊市下限可降至50%

### 二、个股配置策略
1. **持仓数量**：3-5只
2. **单只个股仓位上限**：20%-25%
3. **行业分散要求**：至少覆盖2-3个不同行业

### 三、动态调整规则
1. **止损机制**：单只个股亏损达8%-10%无条件止损
2. **止盈机制**：盈利达20%以上可分批止盈

## 当前代码分析

当前止损/止盈计算逻辑：
```python
stop_loss_price = round(current_price * 0.92, 2)  # 8%止损
take_profit_price = round(current_price * 1.10, 2)  # 10%止盈
```

## 实施步骤

### 步骤1：修改止损止盈计算逻辑
- 止损价格：`current_price * 0.90`（10%止损）
- 止盈价格：`current_price * 1.20`（20%止盈）

### 步骤2：添加仓位控制检查
- 检查单只个股仓位是否超过25%
- 检查总股票仓位是否超过70%
- 检查现金比例是否低于30%

### 步骤3：添加持仓数量限制
- 检查当前持仓数量是否已达到5只

### 步骤4：添加行业分散检查
- 检查当前持仓行业分布
- 提示用户行业集中风险

### 步骤5：更新前端界面
- 显示仓位配置建议
- 显示风险提示信息

# 计划数量与剩余数量逻辑优化计划

## 需求分析

### 核心概念
- **计划数量**：仓位管理中这只股票最多可以买入的数量（固定值）
- **持仓数量**：通过买入/卖出操作后的实际持仓
- **剩余数量**：计划数量 - 持仓数量

### 操作对持仓的影响

| 操作 | 持仓变化 | 剩余变化 |
|------|----------|----------|
| 建仓（买入） | +买入数量 | -买入数量 |
| 加仓（买入） | +买入数量 | -买入数量 |
| 减仓（卖出） | -卖出数量 | +卖出数量 |
| 清仓（卖出） | 归零 | 恢复到计划数量 |

### 计算公式
```
持仓数量 = 累计买入数量 - 累计卖出数量
剩余数量 = 计划数量 - 持仓数量
```

## 当前问题

### 前端当前逻辑
```javascript
const getRemainingQuantity = () => {
  const direction = executeStepForm.value.trade_direction
  const planned = executeStepForm.value.planned_quantity
  const holding = executeStepForm.value.holding_quantity || 0
  
  if (direction === '建仓' || direction === '加仓') {
    return planned  // 错误：应该返回 planned - holding
  } else if (direction === '减仓' || direction === '清仓') {
    return Math.min(planned, holding)  // 错误：应该返回 planned - holding
  }
  return planned
}
```

### 正确逻辑
```javascript
const getRemainingQuantity = () => {
  const planned = executeStepForm.value.planned_quantity
  const holding = executeStepForm.value.holding_quantity || 0
  return planned - holding  // 剩余数量 = 计划数量 - 持仓数量
}
```

## 实施方案

### 前端修改

#### 1. 修改剩余数量计算

```javascript
const getRemainingQuantity = () => {
  const planned = executeStepForm.value.planned_quantity
  const holding = executeStepForm.value.holding_quantity || 0
  return Math.max(0, planned - holding)
}
```

#### 2. 修改成交数量最大值
```javascript
const getMaxQuantity = () => {
  const direction = executeStepForm.value.trade_direction
  const planned = executeStepForm.value.planned_quantity
  const holding = executeStepForm.value.holding_quantity || 0
  const remaining = planned - holding
  
  if (direction === '建仓' || direction === '加仓') {
    return Math.max(0, remaining)  // 最多买入剩余数量
  } else if (direction === '减仓' || direction === '清仓') {
    return holding  // 最多卖出持仓数量
  }
  return 0
}
```

#### 3. 修改成交数量默认值
```javascript
const getDefaultQuantity = (direction, planned, holding) => {
  const remaining = planned - holding
  
  if (direction === '建仓') {
    return Math.min(planned, remaining)  // 建仓：最多买入剩余数量
  } else if (direction === '加仓') {
    return Math.min(planned, remaining)  // 加仓：最多买入剩余数量
  } else if (direction === '减仓') {
    return Math.min(planned, holding)    // 减仓：最多卖出持仓数量
  } else if (direction === '清仓') {
    return holding                        // 清仓：卖出全部持仓
  }
  return 0
}
```

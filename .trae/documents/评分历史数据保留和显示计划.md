# 评分历史数据保留和显示计划

## 需求分析

### 当前情况
1. 评分技能使用 `INSERT OR REPLACE`，同一天同一股票的评分会被替换
2. 不同天的评分会被保留（历史数据已保留）
3. 前端页面显示了评分次数 `score_count`
4. 但没有显示历次打分各指标变化情况

### 期望
1. 保留历史评分数据（已实现）
2. 页面显示打分次数（已实现）
3. 新增：显示历次打分各指标变化情况

## 数据结构

### 历史评分记录

| 字段 | 说明 |
|------|------|
| score_date | 评分日期 |
| total_score | 总分 |
| rating | 评级 |
| ma_score | 均线得分 |
| macd_score | MACD得分 |
| volume_score | 成交量得分 |
| bollinger_score | 布林线得分 |
| obv_score | OBV得分 |
| close_price | 收盘价 |

# 交易记录添加合计计划

## 需求分析

### 当前情况
- 交易记录表格显示每笔交易记录
- 没有合计行显示汇总信息

### 期望
- 在交易记录表格底部添加合计行
- 显示买入金额合计、卖出金额合计、净盈亏等

---

## 实施方案

### 前端修改

文件：`frontend/src/views/TradeExecution.vue`

使用Element Plus的`show-summary`和`summary-method`属性：

```vue
<el-table 
  :data="filteredRecords" 
  style="width: 100%" 
  v-loading="loading"
  show-summary
  :summary-method="getSummaries"
>
```

添加合计方法：
```javascript
const getSummaries = (param) => {
  const { columns, data } = param
  const sums = []
  columns.forEach((column, index) => {
    if (index === 0) {
      sums[index] = '合计'
      return
    }
    if (column.property === 'trade_quantity') {
      const values = data.map(item => item.trade_quantity || 0)
      sums[index] = values.reduce((prev, curr) => prev + curr, 0)
    } else if (column.property === 'trade_amount') {
      const buyAmount = data.filter(item => item.trade_type === '买入')
        .reduce((prev, curr) => prev + (curr.trade_amount || 0), 0)
      const sellAmount = data.filter(item => item.trade_type === '卖出')
        .reduce((prev, curr) => prev + (curr.trade_amount || 0), 0)
      sums[index] = `买入: ${buyAmount.toFixed(2)} | 卖出: ${sellAmount.toFixed(2)}`
    } else {
      sums[index] = ''
    }
  })
  return sums
}
```

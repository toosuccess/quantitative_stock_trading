# 股票池批量评价端到端测试计划

## 测试目标

对股票池中的38只股票进行批量评价测试，验证数据获取是否正常。

## 股票池列表

共38只股票，包括：
- 沪市股票（6开头）：中石油、中海油服、海天精工等
- 深市股票（0开头）：蓝焰控股、万泽股份、大族激光等
- 创业板股票（3开头）：龙磁科技、行云科技、潜能恒信等
- 科创板股票（688开头）：天德钰
- 北交所股票（8开头）：科力股份

## 测试步骤

### 步骤1：批量评价测试
- 对所有38只股票发起评价请求
- 记录每只股票的评价结果
- 验证股价和技术得分是否正常

### 步骤2：结果验证
- 检查技术得分是否为0（异常）
- 检查股价是否为0（异常）
- 统计成功和失败的数量

### 步骤3：问题分析
- 对于获取失败的股票，分析原因
- 检查是否为接口问题或数据问题

### 步骤4：清理临时文件

## 预期结果

- 所有股票的评价功能正常工作
- 股价和技术得分正常获取
- 无异常数据（技术得分=0或股价=0）

# 交易记录合计显示修复计划

## 问题分析

### 当前逻辑
```javascript
if (column.property === 'trade_quantity') {
  const values = data.map(item => item.trade_quantity || 0)
  sums[index] = values.reduce((prev, curr) => prev + curr, 0)
}
```

当前合计显示的是：所有成交数量的累计（买入+卖出）

### 期望逻辑
合计应该显示：当前持仓股数（买入 - 卖出）

---

## 解决方案

### 修改合计计算逻辑
```javascript
if (column.property === 'trade_quantity') {
  const buyQty = data.filter(item => item.trade_type === '买入')
    .reduce((prev, curr) => prev + (curr.trade_quantity || 0), 0)
  const sellQty = data.filter(item => item.trade_type === '卖出')
    .reduce((prev, curr) => prev + (curr.trade_quantity || 0), 0)
  sums[index] = buyQty - sellQty  // 当前持仓股数
}
```

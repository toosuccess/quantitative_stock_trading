## 优化内容

### 1. 添加加仓/减仓步骤

| 优化项 | 优化前 | 优化后 |
|--------|--------|--------|
| 步骤类型 | 建仓、止损清仓、止盈清仓 | 建仓、加仓、减仓、止损清仓、止盈清仓 |
| 自动创建 | 3个步骤 | 5个步骤（含加仓、减仓） |
| 手动添加 | 不支持 | 支持手动添加加仓/减仓步骤 |

### 2. 执行步骤保存数据库

| 优化项 | 优化前 | 优化后 |
|--------|--------|--------|
| 执行记录 | 只创建交易记录 | 创建交易记录+更新步骤状态 |
| 数量更新 | 不更新 | 更新已执行数量、剩余数量 |
| 计划更新 | 不更新 | 更新计划实际数量 |
| 状态判断 | 不判断 | 自动判断步骤是否完成 |

### 执行流程验证

```
1. 执行计划时自动创建步骤：
   - 建仓步骤
   - 加仓步骤（新增）
   - 减仓步骤（新增）
   - 止损清仓步骤
   - 止盈清仓步骤

2. 手动添加步骤：
   - 选择执行中的计划
   - 选择加仓/减仓
   - 输入目标价格和数量
   - 确认添加

3. 执行步骤：
   - 输入成交价格和数量
   - 创建交易记录
   - 更新步骤已执行数量
   - 判断步骤是否完成
   - 更新计划实际数量
```

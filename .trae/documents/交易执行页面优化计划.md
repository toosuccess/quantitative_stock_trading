# 交易执行页面优化计划

## 需求分析

### 问题1：添加"止损清仓"交易方向

**当前情况**：

* 现在有“建仓/加仓/减仓/清仓“执行步骤，需要添加"止损清仓"执行步骤。

### 问题2：交易计划下拉没有数据

**当前情况**：

* loadData中调用`planApi.getPlanList('ACC001')`

* 但实际账户ID是"ACCBBF480B9"

* 导致planOptions为空

**修复方案**：

* 修改为不传account\_id，获取所有计划

* 或使用正确的account\_id

### 问题3：目标价格和数量默认值

**当前情况**：

* 目标价格默认为0

* 数量默认为0

**期望逻辑**：

* 目标价格：读取股票当前价格

* 数量：默认为剩余股票数量的一半

### 问题3修复：目标价格和数量默认值

文件：`frontend/src/views/TradeExecution.vue`

```javascript
const showAddStepDialog = () => {
  const selectedPlan = filters.value.planId || (executingPlans.value[0]?.plan_id || '')
  const plan = allPlans.value.find(p => p.plan_id === selectedPlan)
  
  // 获取股票当前价格
  const currentPrice = plan?.current_price || plan?.target_price || 0
  
  // 计算剩余数量
  const plannedQty = plan?.planned_quantity || 0
  const holdingQty = plan?.holding_quantity || 0
  const remainingQty = plannedQty - holdingQty
  
  addStepForm.value = {
    plan_id: selectedPlan,
    trade_direction: '加仓',
    target_price: currentPrice,  // 使用当前价格
    planned_quantity: Math.floor(remainingQty / 2),  // 剩余数量的一半
    reason: ''
  }
  addStepDialogVisible.value = true
}
```

# 创建计划页面优化计划

## 需求分析

### 1. 加载股票单价价格
- 从股票池点击"创建计划"时，需要自动加载股票的当前价格
- 当前价格应该从score_record表的close_price字段获取
- 避免调用实时行情API导致超时

### 2. 点击创建计划后取消，页面转圈问题
- 点击"创建计划"后，页面会调用fetchCurrentPrice获取实时价格
- 如果API调用超时，会导致页面卡住
- 需要添加超时处理和loading状态

---

## 问题分析

### 问题1：价格获取逻辑
当前代码：
```javascript
const fetchCurrentPrice = async () => {
  if (planForm.value.stock_code) {
    try {
      const data = await stockApi.getRealtime(planForm.value.stock_code)
      // 这个API可能很慢或超时
    } catch (error) {
      console.log('获取实时价格失败，使用默认价格')
    }
  }
}
```

问题：
- `stockApi.getRealtime`调用外部API，可能很慢
- 没有从score_record获取历史收盘价作为备选

### 问题2：对话框打开时卡住
当前代码：
```javascript
onMounted(async () => {
  await loadAccountList()
  loadPlanList()
  if (route.query.stock_code) {
    showCreateDialog()  // 打开对话框
    // 对话框内会调用fetchCurrentPrice
  }
})
```

问题：
- `fetchCurrentPrice`没有超时限制
- 没有loading状态显示
- 用户点击取消后，API调用仍在进行

## 实施方案

### 方案1：从股票池传递价格
修改StockSelection.vue的createPlan方法，传递close_price：
```javascript
const createPlan = (row) => {
  router.push({
    path: '/trade-plan',
    query: { 
      stock_code: row.stock_code, 
      stock_name: row.stock_name,
      close_price: row.close_price  // 添加价格参数
    }
  })
}
```

### 方案2：TradePlan.vue接收价格
修改TradePlan.vue的showCreateDialog方法：
```javascript
const showCreateDialog = () => {
  planForm.value = {
    // ...
    current_price: route.query.close_price || 0,  // 使用传递的价格
  }
  dialogVisible.value = true
  // 不再调用fetchCurrentPrice
}
```

### 方案3：添加loading状态和超时处理
```javascript
const priceLoading = ref(false)

const fetchCurrentPrice = async () => {
  if (!planForm.value.stock_code) return
  
  priceLoading.value = true
  try {
    // 添加超时控制
    const controller = new AbortController()
    const timeoutId = setTimeout(() => controller.abort(), 5000)
    
    const data = await stockApi.getRealtime(planForm.value.stock_code)
    clearTimeout(timeoutId)
    
    if (data && data.price) {
      planForm.value.current_price = data.price
      calculatePrices()
    }
  } catch (error) {
    console.log('获取实时价格失败')
  } finally {
    priceLoading.value = false
  }
}
```

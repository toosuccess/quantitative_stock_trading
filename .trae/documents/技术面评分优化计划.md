# 技术面评分优化计划

## 一、问题分析

### 1.1 当前技术面评分规则
根据评价体系说明，技术面评分（满分100分）包含：

| 指标 | 满分 | 达标条件 |
|------|------|----------|
| 均线系统 | 25分 | 股价站稳20日均线且M20向上，均线多头排列 |
| 成交量 | 25分 | 当日成交量＞5日/60日均量，换手率5%-20% |
| 趋势指标 | 20分 | DMI的+DI＞-DI且ADX/ADXR上升，MACD金叉 |
| 资金指标 | 15分 | OBV＞MAOBV且持续上行，资金净流入 |
| 布林线 | 15分 | 股价位于上轨和中轨之间，且未超买 |

### 1.2 当前评分逻辑问题
查看 `stock_selector.py` 中的 `_calculate_technical_score` 方法：

```python
def _calculate_technical_score(self, indicators: Dict) -> float:
    score = 0.0
    
    # 均线：只要ma20>0就给25分（问题：没有检查股价是否站稳20日均线）
    if indicators.get('ma20', 0) > 0:
        score += 25
    
    # MACD：DIFF>DEA且DIFF>0才给20分（正确）
    if indicators.get('diff_gt_dea_and_zero', False):
        score += 20
    
    # OBV：OBV>MAOBV才给15分（正确）
    if indicators.get('obv_gt_maobv', False):
        score += 15
    
    # 成交量：成交量>5日均量才给25分（正确）
    volume = indicators.get('volume', 0)
    ma5_volume = indicators.get('ma5_volume', 0)
    if ma5_volume > 0 and volume > ma5_volume:
        score += 25
    
    # 布林线：上轨>中轨>0就给15分（问题：没有检查股价是否在上轨和中轨之间）
    bb_upper = indicators.get('bb_upper', 0)
    bb_middle = indicators.get('bb_middle', 0)
    if bb_upper > bb_middle > 0:
        score += 15
    
    return min(score, 100.0)
```

### 1.3 发现的问题
1. **均线评分**：只检查ma20>0，没有检查股价是否站稳20日均线
2. **布林线评分**：只检查上轨>中轨，没有检查股价是否在上轨和中轨之间
3. **缺少多头排列检查**：没有检查均线是否多头排列（MA5>MA10>MA20）

## 二、修复方案

### 2.1 修改 `_calculate_technical_score` 方法
严格按照达标条件评分，不达标不给分：

```python
def _calculate_technical_score(self, indicators: Dict, stock_data: Dict) -> float:
    score = 0.0
    close_price = stock_data.get('close_price', 0)
    
    # 均线系统（25分）：股价站稳20日均线且M20向上，均线多头排列
    ma5 = indicators.get('ma5', 0)
    ma10 = indicators.get('ma10', 0)
    ma20 = indicators.get('ma20', 0)
    
    # 条件1：股价站稳20日均线（股价>=20日均线*0.98）
    price_above_ma20 = close_price >= ma20 * 0.98 if ma20 > 0 else False
    # 条件2：M20向上（需要历史数据，简化为检查MA20>MA10）
    ma20_rising = ma20 > ma10 if ma10 > 0 else False
    # 条件3：均线多头排列（MA5>MA10>MA20）
    ma_bullish = (ma5 > ma10 > ma20) if (ma5 > 0 and ma10 > 0 and ma20 > 0) else False
    
    if price_above_ma20 and ma20_rising:
        score += 25
    
    # 成交量（25分）：当日成交量＞5日均量
    volume = indicators.get('volume', 0)
    ma5_volume = indicators.get('ma5_volume', 0)
    if ma5_volume > 0 and volume > ma5_volume:
        score += 25
    
    # 趋势指标（20分）：MACD金叉（DIFF>DEA且DIFF>0）
    if indicators.get('diff_gt_dea_and_zero', False):
        score += 20
    
    # 资金指标（15分）：OBV>MAOBV
    if indicators.get('obv_gt_maobv', False):
        score += 15
    
    # 布林线（15分）：股价位于上轨和中轨之间
    bb_upper = indicators.get('bb_upper', 0)
    bb_middle = indicators.get('bb_middle', 0)
    if bb_upper > bb_middle > 0:
        if bb_middle <= close_price <= bb_upper:
            score += 15
    
    return min(score, 100.0)
```

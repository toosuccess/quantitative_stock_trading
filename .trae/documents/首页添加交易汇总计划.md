# 首页添加交易汇总计划

## 需求分析

### 当前情况
- 首页显示评分股票、交易计划、交易胜率、总盈亏四个统计卡片
- 没有显示买入和卖出的汇总信息

### 期望
- 添加合计功能
- 根据买入和卖出进行汇总
- 根据已完成状态进行汇总

## 汇总内容

| 项目 | 说明 |
|------|------|
| 总买入金额 | 所有买入交易的金额总和 |
| 总卖出金额 | 所有卖出交易的金额总和 |
| 买入次数 | 买入交易次数 |
| 卖出次数 | 卖出交易次数 |
| 已完成计划 | 已完成状态的交易计划数量 |
| 净盈亏 | 卖出金额 - 买入金额 |

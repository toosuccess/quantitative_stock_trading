# 添加基本面评分和评价体系展示计划

## 一、需求分析

### 1.1 当前评分体系
当前只有技术面评分（满分100分）：
- 均线系统：25分
- 成交量：25分
- 趋势指标：20分
- 资金指标：15分
- 布林线：15分

### 1.2 需要添加的内容
1. **基本面评分**（满分100分）
2. **消息面评分**（满分100分）
3. **政策面评分**（满分100分）
4. **减项扣分**（负分制）
5. **评价体系展示**（在评分详情页面）

---

## 二、评分体系设计

### 2.1 技术面评分（满分100分）- 已有

| 指标 | 满分 | 评分标准 |
|------|------|----------|
| 均线系统 | 25分 | 股价站稳20日均线且M20向上，均线多头排列 |
| 成交量 | 25分 | 当日成交量＞5日/60日均量，换手率5%-20% |
| 趋势指标 | 20分 | DMI的+DI＞-DI且ADX/ADXR上升，MACD金叉 |
| 资金指标 | 15分 | OBV＞MAOBV且持续上行，资金净流入 |
| 布林线 | 15分 | 股价位于上轨和中轨之间，且未超买 |

### 2.2 基本面评分（满分100分）

| 指标 | 满分 | 评分标准 |
|------|------|----------|
| 市盈率(PE) | 15分 | PE<20: 15分, PE<30: 10分, PE<50: 5分, PE>=50: 0分 |
| 市净率(PB) | 15分 | PB<2: 15分, PB<3: 10分, PB<5: 5分, PB>=5: 0分 |
| 净利润增长率 | 20分 | >30%: 20分, >15%: 15分, >0%: 10分, <=0%: 0分 |
| 营收增长率 | 15分 | >20%: 15分, >10%: 10分, >0%: 5分, <=0%: 0分 |
| ROE(净资产收益率) | 20分 | >20%: 20分, >15%: 15分, >10%: 10分, >5%: 5分, <=5%: 0分 |
| 负债率 | 15分 | <40%: 15分, <60%: 10分, <80%: 5分, >=80%: 0分 |

### 2.3 消息面评分（满分100分）

| 指标 | 满分 | 评分标准 |
|------|------|----------|
| 行业利好 | 30分 | 行业政策支持、行业景气度提升 |
| 公司利好 | 30分 | 业绩预告、中标、合作等利好消息 |
| 机构评级 | 20分 | 券商研报评级、目标价 |
| 资金动向 | 20分 | 北向资金、机构持仓变化 |

### 2.4 政策面评分（满分100分）

| 指标 | 满分 | 评分标准 |
|------|------|----------|
| 国家政策 | 30分 | 国家级政策支持（如新能源、芯片等） |
| 地方政策 | 20分 | 地方政府补贴、产业园区政策 |
| 行业规划 | 30分 | 行业发展规划、标准制定 |
| 监管环境 | 20分 | 监管宽松度、合规风险 |

### 2.5 减项扣分（负分制）

| 减项 | 扣分 | 触发条件 |
|------|------|----------|
| 高质押率 | -10分 | 质押比例>30% |
| 减持公告 | -15分 | 近期有大股东减持 |
| 诉讼风险 | -20分 | 存在重大诉讼 |
| 财务异常 | -15分 | 财务数据异常（如应收账款激增） |
| 限售解禁 | -10分 | 近期有限售股解禁 |
| 负面新闻 | -10分 | 存在负面新闻报道 |

---

## 三、综合评分计算

### 3.1 计算公式
```
综合评分 = 技术面得分 × 0.3 + 基本面得分 × 0.3 + 消息面得分 × 0.2 + 政策面得分 × 0.2 + 减项扣分
```

### 3.2 评级标准
- 90-100分：强烈推荐 🔥
- 70-89分：推荐 👍
- 50-69分：中性 😐
- 30-49分：观望 ⚠️
- 0-29分：不推荐 ❌

# 技能模块说明

## 概述

技能模块是量化交易系统的核心组件，负责股票筛选和评价。包含两个主要技能：
1. **选股技能** - 根据策略筛选股票
2. **评价技能** - 对股票进行多维度评分

## 文件结构

```
skills/
├── stock_selector_skills.py    # 选股技能
└── stock_evaluator_skills.py   # 评价技能
```

## 一、选股技能 (stock_selector_skills.py)

### 功能概述
根据十五五规划，筛选符合国家战略方向的股票，动态扩充核心股票池。

### 核心类：StockSelectorSkills

```python
class StockSelectorSkills:
    def __init__(self, db_path=None):
        self.db_path = db_path or DB_PATH
        self.conn = None
```

### 主要方法

#### 1. connect() / disconnect()
连接/断开数据库

#### 2. select_stocks_by_policy()
根据政策导向筛选股票

**返回**：符合政策导向的股票列表

**筛选逻辑**：
1. 获取所有A股股票列表
2. 根据行业匹配政策导向
3. 排除ST股票
4. 保存到stock_basic_info表

#### 3. 政策导向行业配置

```python
POLICY_ORIENTED_INDUSTRIES = {
    '新能源': {'industry_type': '战略性新兴产业', 'growth_potential': '高'},
    '新材料': {'industry_type': '战略性新兴产业', 'growth_potential': '高'},
    '6G': {'industry_type': '未来产业', 'growth_potential': '极高'},
    '核聚变': {'industry_type': '未来产业', 'growth_potential': '极高'},
    '消费升级': {'industry_type': '民生消费', 'growth_potential': '中高'},
    '半导体': {'industry_type': '战略性新兴产业', 'growth_potential': '高'},
    '人工智能': {'industry_type': '未来产业', 'growth_potential': '极高'},
    '生物医药': {'industry_type': '战略性新兴产业', 'growth_potential': '高'},
    '高端制造': {'industry_type': '战略性新兴产业', 'growth_potential': '高'},
    '数字经济': {'industry_type': '未来产业', 'growth_potential': '极高'}

```

### 使用示例

```python

selector = StockSelectorSkills()
selector.connect()

# 执行选股
stocks = selector.select_stocks_by_policy()
print(f"筛选出 {len(stocks)} 只股票")

selector.disconnect()
```

## 二、评价技能 (stock_evaluator_skills.py)

### 功能概述
对stock_basic_info表中的股票进行多维度评分，输出到score_record表。

### 核心类：StockEvaluatorSkills

```python
class StockEvaluatorSkills:
    def __init__(self, db_path=None):
        self.db_path = db_path or DB_PATH
        self.conn = None
```

### 主要方法

#### 1. connect() / disconnect()
连接/断开数据库

#### 2. evaluate_stock(stock_code)
评价单只股票

**参数**：
- `stock_code`: 股票代码（如：sz000534）

**返回**：
```python
{
    'stock_code': 'sz000534',
    'stock_name': '万泽股份',
    'technical_score': 100,
    'fundamental_score': 40,
    'news_score': 0,
    'policy_score': 8,
    'deduction_score': 0,
    'composite_score': 84,
    'rating': '推荐',
    'technical_detail': {...},
    'fundamental_detail': {...},
    'news_detail': {...},
    'policy_detail': {...},
    'deduction_detail': {...}
}
```

#### 3. get_stock_price_data(stock_code)
获取股票价格数据

**数据来源**：akshare `stock_zh_a_hist`

**返回数据**：
- 收盘价、最高价、最低价、成交量
- 均线：MA5、MA10、MA20、MA60
- MACD：DIFF、DEA、MACD
- RSI指标
- OBV指标
- 布林带：上轨、中轨、下轨

#### 4. calculate_technical_score(stock_code)
计算技术面评分

**评分维度**：
| 指标 | 满分 |
|------|------|
| 均线系统 | 25分 |
| 成交量 | 25分 |
| MACD | 20分 |
| OBV | 15分 |
| 布林带 | 15分 |

#### 5. get_fundamental_data(stock_code)
获取基本面数据

**数据来源**：
1. akshare `stock_individual_info_em` - 个股信息
2. akshare `stock_financial_analysis_indicator` - 财务指标
3. akshare `stock_zh_a_spot_em` - 行情数据
4. akshare `stock_financial_report_sina` - 财务报表

**返回数据**：
- PE（市盈率）
- PB（市净率）
- ROE（净资产收益率）
- 净利润增长率
- 营收增长率
- 负债率
- 销售毛利率
- 销售净利率
- 流动比率
- 资产周转率
- 现金流动比率

#### 6. calculate_fundamental_score(stock_code)
计算基本面评分 - 新5大类评分体系

**评分维度**：
| 类别 | 指标 | 满分 |
|------|------|------|
| 盈利能力（30分） | ROE | 20分 |
| 盈利能力（30分） | 净利率 | 10分 |
| 成长能力（25分） | 净利润增长率 | 15分 |
| 成长能力（25分） | 营收增长率 | 10分 |
| 估值（20分） | PE | 12分 |
| 估值（20分） | PB | 8分 |
| 财务健康（15分） | 负债率 | 10分 |
| 财务健康（15分） | 流动比率 | 5分 |
| 现金流&运营（10分） | 资产周转率 | 5分 |
| 现金流&运营（10分） | 现金流动比率 | 5分 |

#### 7. calculate_news_score(stock_code)
计算消息面评分

**数据来源**：akshare `stock_news_em`

**返回数据**：
```python
{
    'events': [
        {
            'date': '2026-02-12',
            'type': '中性',
            'title': '新闻标题',
            'url': 'http://...',
            'source': '证券时报网',
            'summary': '新闻摘要...',
            'score_impact': 0
        }
    ]
}
```

#### 8. calculate_policy_score(stock_code)
计算政策面评分

**数据来源**：
- 股票行业信息
- `POLICY_ORIENTED_INDUSTRIES` 配置

**评分规则**：
| 行业类型 | 得分 |
|---------|------|
| 未来产业 | +10分 |
| 战略性新兴产业 | +8分 |
| 民生消费 | +5分 |
| 其他行业 | +3分 |

#### 9. calculate_deduction_score(stock_code)
计算减项扣分

**检查项**：
- 高质押率：-10分
- 减持公告：-15分
- 诉讼风险：-20分
- 财务异常：-15分
- 限售解禁：-10分

#### 10. save_score_record(result)
保存评分记录到数据库

**保存表**：score_record

### 使用示例

```python
from skills.stock_evaluator_skills import StockEvaluatorSkills

evaluator = StockEvaluatorSkills()
evaluator.connect()

# 评价单只股票
result = evaluator.evaluate_stock('sz000534')
print(f"综合评分: {result['composite_score']}")
print(f"评级: {result['rating']}")

evaluator.disconnect()
```

### 批量评价示例

```python
import sqlite3

conn = sqlite3.connect('trading_system.db')
cursor = conn.cursor()

cursor.execute("SELECT stock_code FROM stock_basic_info")
stocks = cursor.fetchall()

evaluator = StockEvaluatorSkills()
evaluator.connect()

for stock in stocks:
    result = evaluator.evaluate_stock(stock[0])
    print(f"{stock[0]}: {result['composite_score']}分 - {result['rating']}")

evaluator.disconnect()
conn.close()
```

## 三、数据流

```
选股技能
    ↓

    ↓
评价技能
    ↓
score_record表（评分记录）
    ↓
前端展示
```

## 五、配置说明

### 数据库路径
默认路径：`backend/database/trading_system.db`

### 政策导向配置
修改 `POLICY_ORIENTED_INDUSTRIES` 字典可调整行业分类和评分权重。

### 评分权重调整
修改各评分方法中的评分规则可调整评分权重。
